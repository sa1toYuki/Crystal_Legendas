# Como Contribuir

Este projeto é colaborativo e toda ajuda é muito bem-vinda.

## Como contribuir com legendas (.ass)

### 1. Fork e Clone o repositório
- Faça um **Fork** deste repositório
- Clone o seu fork para o seu computador

### 2. Edição de arquivos .ass

Os arquivos de legenda neste repositório estão no formato **Advanced SubStation Alpha (.ass)**.

**Ao editar ou criar uma legenda .ass, siga estas recomendações:**

- Mantenha a consistência de estilo com as outras legendas do repositório
- Mantenha a formatação padrão da fansub
- Use borda e sombra adequadas para boa visibilidade
- Respeite as regras básicas de timing (não sobrepor legendas, tempo mínimo de exibição)
- Corrija os possíveis erros de tradução, digitação, pontuação e sincronia
- Mantenha os estilos existentes (`Default`, `Italics`, etc.) sempre que possível
- Não remova os metadados de autoria no cabeçalho do arquivo

**Dica:** Recomendamos usar o programa **Aegisub** para editar os arquivos .ass.

### 3. Envie suas alterações

1. Após editar o arquivo, faça um **commit** com uma mensagem clara

2. Faça **Push** para o seu fork

3. Abra um **Pull Request** para a branch `main` deste repositório

### Processo de Revisão

- A equipe irá analisar o Pull Request
- Poderemos solicitar ajustes ou melhorias
- Após aprovação, o arquivo será incorporado ao repositório
- **Seu nome de usuário (ou nome real, se preferir) será adicionado aos créditos** da legenda e/ou no arquivo de agradecimentos

---

## Créditos

Contribuidores aprovados terão seu nome listado em:

- No cabeçalho do arquivo `.ass` (campo `Original Script` ou `Author`)
- Nos créditos do EPISÓDIO
- No arquivo `CREDITOS.md` (quando criado)

---

**Qualquer dúvida?**  
Abra uma [Issue](https://github.com/sa1toYuki/Crystal_Legendas/issues) ou entre em contato.
