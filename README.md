# Legendas Open

Repositório aberto dedicado ao compartilhamento de legendas em português.
## Sobre o Projeto

Este repositório tem como principal objetivo **compartilhar legendas** de forma livre e colaborativa para os animes traduzidos pela Crystal Fansub.

Todas as legendas disponíveis neste repositório estão licenciadas sob **Creative Commons Atribuição 4.0 Internacional (CC BY 4.0)**. Isso significa que você pode usar, modificar, distribuir, desde que dê os devidos créditos.

## Como Contribuir

Este é um repositório **colaborativo**! Você é livre e muito bem-vindo para contribuir:

- Corrigir erros de tradução, sincronia ou ortografia
- Melhorar legendas existentes
- Adicionar legendas em outros idiomas
- Sugerir melhorias

### Forma mais fácil de contribuir:
1. Faça um **Fork** deste repositório
2. Faça as alterações desejadas
3. Abra um **Pull Request** descrevendo as mudanças

Toda contribuição será analisada e, se estiver dentro dos padrões do projeto, será incorporada.

## Estrutura do Repositório

- `index` → Pasta principal com todas as legendas organizadas
- `LICENSE.md` → Licença completa (CC BY 4.0)

## Créditos

As legendas presentes aqui foram criadas e revisadas por diversos colaboradores da comunidade. Cada arquivo contém informações de autoria quando aplicável.

---

**Sinta-se em casa!**  
Quanto mais pessoas contribuírem, melhor será o repositório para todos.

Qualquer dúvida ou sugestão, abra uma **Issue** ou um **Pull Request**.
